Migrating Hello Address to Flutter for smoother cross-platform real estate UX

Hello Address needed a Flutter migration that made the app feel effortless and more stable. I worked on the migration with Firebase, Riverpod, token refresh handling, stored session data, source-based navigation, and smoother app flows across property, detail, onboarding, and dashboard journeys.

Benefit

Challenge

Resolution

Benefit

This project is presented as a migration and app-flow reliability case study. The main engineering value was not only moving screens to Flutter, but making the migrated product easier to use, less crash-prone, and more consistent across Android and iOS.
